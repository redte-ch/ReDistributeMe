# First Iteration

1. Create a new project

    ```sh
    npm create astro@latest
    cd redistribute-me
    npm run dev
    ```
    
2. Open [localhost:4321](http://localhost:4321/) in your browser.

3. Create a new git repository and commit your project

    ```sh
    git init
    git add .
    git commit -m "chore: initial commit"
    ```
